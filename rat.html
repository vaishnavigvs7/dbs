Customer Rating System

HIGH MEDIUM LOW
H1: If there are more than 10 incoming or outgoing (INN | OUT) transaction in a month from high risk country. M1: If there more than 6 incoming or outgoing (INN | OUT) transaction in a month from high risk/Non-high country. L1: If there less than 10 incoming or outgoing (INN | OUT) transaction in a month from non-high-risk country.
H2: If sum of all incoming(INN) transaction is more than 1000$ in a month. M2: If sum of all incoming(INN) transaction is more than 600$ and less than 1000$ in a month. L2: If sum of all incoming(INN) transaction is less than 600$ in a month.
H3: If sum of all outgoing (OUT) transaction is more than 800$ in a month. M3: If sum of all outgoing (OUT) transaction is more than 500$ and less than 800$ in a month. L3: If sum of all outgoing (OUT) transaction is less than 500$ in a month.
H4: If number of transaction in a day is more than 20. M4: If number of transaction in a day is more than 10 but less than 20. L4: If number of transaction in a day is less than 10.
